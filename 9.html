What is the server?

  • The machine that your browser talks to

  • Generally "speaks" http

  • Serves you static html + files

  • Also can generate "dynamic content"

  • But, by the time it gets to your browser, it looks like static content.

Using Heroku

  • Download and install the heroku toolbelt

  • Create a repository on GitHub

  • Make sure you have an index.php file

  • Run: heroku create app-name

  • Run: git push heroku master

  • Go to: app-name.herokuapp.com

  • Bonus points: Configure your own domain

SSH

PHP

  • The best first language to learn

  • Available on almost all hosting accounts

  • Script and HTML mixed together in one file

  • Not the only language (not my favorite language)

PHP 101

  • Sort of object-oriented

  • Variables prefixed with “$”

  • Lots of superficial similarities to javascript

  • Embed into HTML

  • http://www.learnphp-tutorial.com/

If you continue with development...

  • You’ll want to set up your computer as a development machine

  • http://foundationphp.com/tutorials/php_leopard.php

  • http://php.iis.net/

Exercise 2: Hello World

  • Log into your ftp (or my) account.

  • Create a new file (locally) called index.php

  • Type "Hello World" into the file

  • Delete the index.html file you uploaded before

  • Upload your index.php

  • Go to the web page.

Exercise 3: Hello PHP

  • Change your file from "Hello World" to

  • < ?php echo "Hello World" ?>

  • then try:

  • < ?php echo “Hello World at “ . time(); ?>

Forms

  • Data submitted to the server

  • Accessible in php via $_REQUEST, $_POST, and $_GET

  • e.g.
    < form action=’?’ method=‘post’>
    < input type=‘text’ name=‘field’/>
    < input type=‘submit’ value=’Go!’/> </form>

Exercise 4: Create a form

  • Accept and input first and last name

  • Output them to the page with <?php print_r($_REQUEST); ?>

Including the same content

  • What about the same header and footer over and over on your site?

  • use the php include(..) method

  • Can also use require_once(..)

Exercise 5

  • Modify your page to include a header and a footer that you include with PHP

  • Then create another page that uses the same header and footer

Setting variables

  • You can set variables with = sign assignment, e.g.: $page = "hello"

  • You can access those variables anywhere you like.

  • This includes other files, which means you can use them to pass in state

Exercise 6

  • Create navigation in your header

  • Set a variable to indicate what page you are on

Exercise 7: Sending an Email

  • Create a form that lets people submit a subject and a message

  • Look at the mail command and send an email with a the subject and message

  • http://php.net/manual/en/funcref.php
